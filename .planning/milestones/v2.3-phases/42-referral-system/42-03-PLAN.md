**Update src/stores/referralStore.ts:**

Add to interface:
```typescript
capturedReferralCode: string | null  // Code captured from deep link (pre-signup)
setCapturedCode: (code: string) => void  // Store captured code
clearCapturedCode: () => void
attributeReferral: () => Promise<void>  // Create referral record post-signup
```

Persist `capturedReferralCode` in localStorage (use zustand persist).

`setCapturedCode(code)`:
- Validate code format (contains hyphen, alphanumeric)
- Store in state

`attributeReferral()`:
- Get current user from authStore
- If no capturedReferralCode, return early
- Query profiles to find referrer by referral_code
- If found, insert into referrals table:
  - referrer_id: from profiles query
  - recruit_id: current user id
  - referral_code_used: capturedReferralCode
  - status: 'pending'
- Clear capturedReferralCode after successful insert
- Handle duplicate gracefully (user already referred)

**Update src/lib/deep-link.ts:**

1. Import referralStore: `import { useReferralStore } from '@/stores/referralStore'`

2. Add `/join/` to allowedPaths list

3. In handleDeepLink, before the existing pathname checks, add:
```typescript
// Handle referral deep links
if (parsed.pathname.startsWith('/join/')) {
  const code = parsed.pathname.replace('/join/', '')
  if (code && code.includes('-')) {
    useReferralStore.getState().setCapturedCode(code)
    // Navigate to signup to complete referral
    navigate('/auth/signup')
    return
  }
}
```

This captures the code and redirects to signup flow.
