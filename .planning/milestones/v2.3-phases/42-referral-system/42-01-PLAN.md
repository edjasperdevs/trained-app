Create `src/stores/referralStore.ts` using zustand with persist:

Interface:
```typescript
interface ReferralStore {
  referralCode: string | null
  recruits: Recruit[]
  isLoading: boolean

  // Actions
  fetchReferralCode: () => Promise<string>  // Gets or generates code
  fetchRecruits: () => Promise<void>        // Loads recruits list
  getReferralLink: () => string             // Returns full URL

  // Computed
  pendingCount: () => number
  completedCount: () => number
  totalDPEarned: () => number
}

interface Recruit {
  id: string
  callsign: string | null  // username from profiles
  rank: number
  status: 'pending' | 'completed'
  dpEarned: number
  createdAt: string
  completedAt: string | null
}
```

Implementation:

1. `fetchReferralCode()`:
   - Get current user from authStore
   - Query profiles for referral_code
   - If null, generate new code:
     - Get username (callsign), default to 'USER' if empty
     - Generate 4-char random suffix (A-Z0-9)
     - Format: `{CALLSIGN.toUpperCase()}-{SUFFIX}`
     - Update profiles with new code
   - Return code

2. `getReferralLink()`:
   - Return `https://app.welltrained.fitness/join/${referralCode}`
   - Return empty string if no code

3. `fetchRecruits()`:
   - Query referrals where referrer_id = current user
   - Join with profiles to get recruit username and rank data
   - Map to Recruit interface

4. Helper methods:
   - pendingCount: filter recruits by status === 'pending'
   - completedCount: filter recruits by status === 'completed'
   - totalDPEarned: sum of dpEarned from all recruits

Export from `src/stores/index.ts`.

Use existing patterns from dpStore for Supabase queries.
