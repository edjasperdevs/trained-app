Error Monitoring:
- `grep -n "VITE_SENTRY_DSN" .env.example` (DSN env var documented)

- `grep -rn "captureError" src/lib/sync.ts src/stores/authStore.ts src/lib/foodApi.ts` (Sentry wired in)
- `grep -n "sentrySetUser\|sentryClearUser" src/stores/authStore.ts` (user context wired)

Analytics:
- `grep -n "plausible" index.html` (Plausible script tag)
- `grep -n "trained-app-eta.vercel.app" index.html` (correct domain)

Deployment:
- `cat vercel.json` (Vercel config exists with rewrites)

- Check that no `.env` files are committed: `git ls-files .env*`

Build verification:
- `npx tsc --noEmit` (TypeScript clean)
- `npx vite build` (build succeeds)
- Check bundle output for reasonable sizes
