Interface:
```typescript
interface DailyDP {
  date: string
  training: number   // count of training awards today
  meals: number      // count of meal awards today (cap at 3)
  protein: number    // count of protein awards today
  steps: number      // count of steps awards today
  sleep: number      // count of sleep awards today
  total: number      // total DP earned today
}

interface DPStore {
  totalDP: number
  currentRank: number
  obedienceStreak: number
  longestObedienceStreak: number
  lastActionDate: string | null
  lastCelebratedRank: number
  dailyLogs: DailyDP[]

  awardDP: (action: DPAction, amount?: number) => { dpAwarded: number; rankedUp: boolean; newRank: number }
  getRankInfo: () => { name: string; rank: number; dpForNext: number; progress: number }
  checkObedienceStreak: () => void
  getTodayLog: () => DailyDP | null
  resetDP: () => void
}
```

Key implementation details:
- Use `persist` middleware with `name: 'trained-dp'`
- `awardDP`: Check meal cap (3/day) before awarding. Add to totalDP, calculate new rank, update daily log, call checkObedienceStreak, update lastCelebratedRank if ranked up. Return result object.
- `getRankInfo`: Find current rank entry and next rank entry. Calculate `dpForNext` = next threshold - totalDP. Calculate `progress` = (totalDP - currentThreshold) / (nextThreshold - currentThreshold). At max rank: progress = 1, dpForNext = 0.
- `checkObedienceStreak`: Use `getLocalDateString()` and `getLocalDaysDifference()` from dateUtils. If lastActionDate is today, no-op. If yesterday, increment. If gap > 1 day or null, reset to 1. Update longestObedienceStreak.
- `getTodayLog`: Find daily log entry matching today's date.
- `resetDP`: Set all fields to initial values, clear dailyLogs.
- Export `useDPStore`, `DPAction`, `DailyDP`, `DP_VALUES`, `RANKS` (RANKS needed by UI for rank names).

Update `src/stores/index.ts`: Add `export { useDPStore } from './dpStore'` and `export type { DailyDP, DPAction } from './dpStore'`. Keep existing `useXPStore` export (needed for Phase 24 migration).
