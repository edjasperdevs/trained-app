**Macro calculation (useEffect on mount):**
- Use default values: height=70 inches (5'10"), weight=185 lbs, age=30, gender='male'
- Map onboardingStore.data.goal to macroStore Goal type:
  - build_muscle -> 'bulk'
  - lose_fat -> 'cut'
  - get_stronger -> 'recomp'
  - improve_fitness -> 'maintain'
- Use Mifflin-St Jeor formula (same as macroStore.calculateMacros):
  ```
  BMR = (10 * weightKg) + (6.25 * heightCm) - (5 * age) + 5  // male
  TDEE = BMR * 1.55 (moderate activity)
  Calories = TDEE + goalAdjustment (-500 cut, -200 recomp, 0 maintain, +300 bulk)
  Protein = weight * 1 (1g per lb)
  Fat = calories * 0.27 / 9
  Carbs = (calories - protein*4 - fat*9) / 4
  ```
- Store calculated values in local state: { protein, carbs, fat, calories }

**Donut chart (custom SVG, not Recharts):**
- SVG circle with strokeDasharray for segments
- Three segments: protein (gold #D4A853), carbs (lighter gold #E5C98A), fat (muted #8B7355)
- Center text: calories number (large Oswald font) + "CALORIES" label below
- Animation: use framer-motion to animate strokeDashoffset from full to final over 800ms
- Start animation after 300ms delay (after screen fade-in)

**Stat cards (3 horizontal):**
- Each card: bg-[#26282B], rounded-xl, py-4, text-center
- Label: "PROTEIN" / "CARBS" / "FAT" in text-[#A1A1AA], text-xs, uppercase
- Value: "{X}g" in white, text-2xl, Oswald font
- Percentage: "{Y}%" in text-[#A1A1AA], text-sm
- Count-up animation: start from 0, count to target over 600ms
- Animation starts 800ms after mount (after donut completes)
- Use requestAnimationFrame or setInterval for smooth count-up

**Footer:**
- Text: "These targets are calculated from your profile. Your coach may adjust them."
- text-[#A1A1AA], text-sm, text-center, mb-4

**Bottom CTA:**
