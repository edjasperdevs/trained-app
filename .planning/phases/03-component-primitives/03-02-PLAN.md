```css

button:focus-visible,
a:focus-visible,
input:focus-visible,

textarea:focus-visible,
[role="button"]:focus-visible {
  outline: 2px solid var(--color-primary);
  outline-offset: 2px;
}

/* Inputs already have box-shadow focus, suppress double outline */
.input-base:focus-visible {
  outline: none;
}
```
