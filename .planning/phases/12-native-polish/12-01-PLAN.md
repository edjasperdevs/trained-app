  <action>
Rewrite `src/lib/haptics.ts` to use `@capacitor/haptics` on native platforms and fall back to `navigator.vibrate()` on web. The existing haptic taxonomy (light, medium, success, heavy, error) is already called from 4 active sites -- no call sites need to change.

Replace the entire file with:

```typescript
import { Haptics, ImpactStyle, NotificationType } from '@capacitor/haptics'
import { isNative } from './platform'

const canVibrate = typeof navigator !== 'undefined' && 'vibrate' in navigator

export const haptics = {
  /** Light tap - set completion, toggles */
  light: () => {
    if (isNative()) {
      Haptics.impact({ style: ImpactStyle.Light })
    } else if (canVibrate) {
      navigator.vibrate(10)
    }
  },

  /** Medium tap - action confirmed */
  medium: () => {
    if (isNative()) {
      Haptics.impact({ style: ImpactStyle.Medium })
    } else if (canVibrate) {
      navigator.vibrate(25)
    }
  },

  /** Success pattern - workout complete, check-in, achievement unlock */
  success: () => {
    if (isNative()) {
      Haptics.notification({ type: NotificationType.Success })
    } else if (canVibrate) {
      navigator.vibrate([15, 50, 30])
    }
  },

  /** Heavy tap - important milestone like XP claim */
  heavy: () => {
    if (isNative()) {
      Haptics.impact({ style: ImpactStyle.Heavy })
    } else if (canVibrate) {
      navigator.vibrate(50)
    }
  },

  /** Error buzz - something went wrong */
  error: () => {
    if (isNative()) {
      Haptics.notification({ type: NotificationType.Error })
    } else if (canVibrate) {
      navigator.vibrate([50, 30, 50])
    }
  },
}
```

Key mapping: light/medium/heavy -> ImpactStyle.Light/Medium/Heavy via Haptics.impact(). success/error -> NotificationType.Success/Error via Haptics.notification(). Haptic calls are fire-and-forget (no await) to avoid blocking UI -- the returned Promises resolve silently even on unsupported devices.

Do NOT modify any call sites. The 4 active call sites (Workouts.tsx lines 154+236, Home.tsx line 546, XPClaimModal.tsx line 156) all import from `@/lib/haptics` and call the same methods.
