**1. Add localStorage migration in src/main.tsx (BEFORE React mount):**

Add the following migration code before the `ReactDOM.createRoot()` call, after the Sentry init and font imports but before mounting:

```typescript
// Theme migration: Remove legacy theme selection
// Users who had 'gyg' selected will now get Trained (the only theme)
const legacyTheme = localStorage.getItem('app-theme')
if (legacyTheme) {
  localStorage.removeItem('app-theme')
  // Also remove body class that ThemeProvider used to set
  document.body.classList.remove('theme-trained', 'theme-gyg')
}
```

This runs synchronously on boot, before React hydrates, so there's no flash of wrong theme.
