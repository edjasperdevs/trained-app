<task type="checkpoint:human-action" gate="blocking">
  <name>Task 1: Configure App Store Connect metadata and submit for review</name>
  <action>
This task requires manual action in App Store Connect. Claude cannot perform these steps.

**Step 1: Create a demo account for Apple reviewers**
1. In the WellTrained app (web or TestFlight), create a new account with:
   - Email: `review@welltrained.fitness` (or similar)
   - Password: a secure but sharable password
2. Populate the account with demo data:
   - Add a username and profile settings
   - Log a few workouts, macro entries, and weight entries
   - Ensure the account has some XP and a visible streak
3. Note the credentials -- you will enter them in App Store Connect

**Step 2: Take screenshots**
1. Use Xcode Simulator (iPhone 15 Pro Max for 6.7" display class)
2. Launch the app with the demo account signed in
3. Take screenshots of these screens (minimum 3, recommended 5):
   - Home screen (showing reminders, streaks, XP)
   - Workouts screen (showing a workout plan)
   - Macros screen (showing nutrition targets/logs)
   - Settings/Weight tracking (showing the chart)
   - Weekly Check-in (showing the form)
4. Screenshots must be 1290x2796 pixels (6.7" display) or 1320x2868 (6.9" display)
5. Optional: Use https://shots.so or Figma to add device frames and marketing text

**Step 3: App Store Connect metadata**
1. Go to https://appstoreconnect.apple.com/ > My Apps > WellTrained
2. Under "App Information":
   - Primary category: **Health & Fitness**
   - Secondary category: **Lifestyle** (optional)
   - Content rights: "This app does not contain third-party content"
   - Age rating: 4+ (no objectionable content)

3. Under "Pricing and Availability":
   - Price: Free
   - Available in all territories

4. Under "App Privacy" (privacy nutrition labels):
   Declare the following data types:

   | Category | Data Type | Linked to Identity? | Used for Tracking? | Purpose |
   |----------|-----------|--------------------|--------------------|---------|
   | Contact Info | Email Address | Yes | No | App Functionality |
   | Health & Fitness | Fitness | Yes | No | App Functionality |
   | User Content | Photos or Videos | Yes | No | App Functionality |
   | User Content | Other User Content | Yes | No | App Functionality |
   | Identifiers | User ID | Yes | No | App Functionality |
   | Diagnostics | Crash Data | No | No | Analytics |
   | Diagnostics | Performance Data | No | No | Analytics |
   | Usage Data | Product Interaction | No | No | Analytics |

5. Under "Version Information" (for version 1.0.0):
   - Description (suggested):
     ```
     WellTrained is your personal fitness companion designed to build daily discipline through structured training, nutrition tracking, and gamification.

     FEATURES:
     - Structured workout plans (3, 4, or 5 days per week)
     - Daily macro and nutrition tracking
     - Weight logging with trend analysis and projected goal dates
     - XP system that rewards consistency
     - Achievement badges for milestones
     - Weekly check-in reports with photo progress tracking
     - Coach integration for personalized programming
     - Push notifications for daily reminders and streak protection
     - Dark theme optimized for gym environments

     Structure creates freedom. Track your discipline, see your progress, and stay accountable.
     ```
   - Keywords (100 chars max):
     `fitness,workout,macros,nutrition,tracker,gym,training,discipline,coach,health`
   - Support URL: `https://app.welltrained.fitness`
   - Privacy Policy URL: `https://app.welltrained.fitness/privacy`
   - Upload screenshots from Step 2

6. Under "Review Information":
   - Sign-in required: Yes
   - Demo account username: (from Step 1)
   - Demo account password: (from Step 1)
   - Notes for reviewer:
     ```
     This app uses Capacitor (WKWebView) with deep native integration including:
     - Apple Push Notifications (APNs) for coach actions and daily reminders
     - Local notifications for streak protection and scheduled reminders
     - Haptic feedback (Taptic Engine) on workout completion and XP claims
     - Native share sheet for data export
     - Badge count management
     - Universal Links for password reset deep linking

     The app is a fitness gamification platform where users track workouts,
     nutrition macros, and weight with an XP reward system. Coach features
     allow trainers to assign workouts and review weekly check-ins.

     Demo account has pre-populated workout logs, macro entries, and weight
     history to demonstrate full functionality.
     ```

**Step 4: Submit for review**
1. Select the TestFlight build to submit
2. Click "Add for Review"
3. Click "Submit for Review"
4. Apple review typically takes 24-48 hours (can be faster)

**Step 5: After approval**
1. Once approved, the app status changes to "Ready for Distribution"
2. If auto-release is configured, it goes live immediately
3. If manual release, click "Release This Version"
4. Verify the app is searchable and downloadable on the App Store
