  <action>
Wire handleResetPassword function with Supabase password reset:

**Implementation:**
```typescript
const handleResetPassword = async () => {
  if (!isValidEmail(email) || isLoading) return

  setIsLoading(true)

  const { error } = await supabase!.auth.resetPasswordForEmail(email, {
    redirectTo: 'welltrained://reset-password',
  })

  setIsLoading(false)

  if (error) {
    // For security: don't reveal if email exists
    // Still show success state
  }

  setSubmittedEmail(email)
  setIsSuccess(true)
}
```

**Button wiring:**
- onClick={handleResetPassword}
- Loading state shows "Sending..."
- After success, form transitions to success state

**Notes:**
- Always show success (security best practice - don't reveal if email exists)
- redirectTo URL for deep link back to app when user clicks email link
- Clear any errors on successful submission
