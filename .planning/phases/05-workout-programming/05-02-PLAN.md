**useCoachTemplates.ts** -- Follow the `useClientDetails.ts` pattern (Map cache, direct Supabase queries, dev bypass).

```typescript
// Cache pattern
const templateCache = new Map<string, { data: WorkoutTemplate[]; fetchedAt: number }>()
const CACHE_TTL = 5 * 60 * 1000 // 5 minutes
```

Hook exports:
- `templates: WorkoutTemplate[]` -- cached list
- `isLoading: boolean`
- `fetchTemplates(): Promise<void>` -- queries `workout_templates` WHERE `coach_id = auth.uid()`, ordered by `updated_at DESC`. Dev bypass returns `mockWorkoutTemplates` from devSeed.
- `createTemplate(name: string, exercises: PrescribedExercise[]): Promise<WorkoutTemplate | null>` -- INSERT into `workout_templates`. Returns the created row. Invalidates cache.
- `updateTemplate(id: string, updates: { name?: string; exercises?: PrescribedExercise[] }): Promise<void>` -- UPDATE workout_templates SET ... WHERE id. Invalidates cache.
- `deleteTemplate(id: string): Promise<void>` -- DELETE FROM workout_templates WHERE id. Invalidates cache.
- `assignWorkout(clientId: string, date: string, exercises: PrescribedExercise[], templateId?: string, notes?: string): Promise<{ error: string | null; isOverwrite?: boolean }>` -- UPSERT into assigned_workouts with `onConflict: 'client_id,date'`. Exercises are the snapshot (copied from template at call time). Returns error string or null.
- `checkExistingAssignment(clientId: string, date: string): Promise<AssignedWorkout | null>` -- SELECT from assigned_workouts WHERE client_id AND date. Used to detect conflicts before overwriting.
- `fetchClientAssignments(clientId: string, startDate: string, endDate: string): Promise<AssignedWorkout[]>` -- SELECT from assigned_workouts for the date range. Used by coach to see what's already assigned.

Import types from `@/lib/database.types` (direct import, NOT barrel). Import `getSupabaseClient`, `supabase` from `@/lib/supabase`. Import `isDevBypass` from the same pattern used by useClientRoster/useClientDetails.

For dev bypass mode: return mock data from devSeed, skip Supabase calls.
