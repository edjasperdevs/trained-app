1. Install @capgo/capacitor-health:
   ```bash
   npm install @capgo/capacitor-health@^7.2.15
   npx cap sync ios
   ```

2. Create src/lib/health.ts with these functions (all guarded by isIOS()):

   - `isHealthAvailable(): Promise<boolean>` - Returns false on non-iOS, calls Health.isAvailable() otherwise

   - `requestHealthPermission(): Promise<boolean>` - Returns false on non-iOS, calls Health.requestAuthorization({ read: ['steps', 'sleep'], write: [] })

   - `readTodaySteps(): Promise<number>` - Returns 0 on non-iOS, uses Health.queryAggregated with dataType: 'steps', aggregation: 'sum', bucket: 'day' for today (midnight to now)

   - `readTodaySleep(): Promise<number>` - Returns 0 on non-iOS, uses Health.readSamples with dataType: 'sleep' for last night (6pm yesterday to noon today), filters for asleep states only ('asleep', 'asleepCore', 'asleepDeep', 'asleepREM', 'asleepUnspecified'), sums durations, returns total minutes

Pattern: Follow src/lib/push.ts for try/catch structure and platform guard pattern.
  </action>
  <verify>
    <automated>npx tsc --noEmit && npm test -- --run</automated>
  </verify>
  <done>health.ts exports all 4 functions, TypeScript compiles without errors, no runtime crash on web (isIOS guard protects plugin calls)</done>
</task>
