  <action>
Create `src/components/CoachGuard.tsx` as a route guard component.

Requirements:
1. Import `isCoach` from `@/lib/supabase` (direct import, NOT from barrel — matches existing pattern in sync.ts and SyncStatusIndicator)
2. Import `Navigate` from `react-router-dom` for redirect
3. Use `useState` + `useEffect` to call `isCoach()` on mount
4. Three states: `loading`, `authorized`, `unauthorized`
5. While loading: render a centered loading spinner matching the existing auth loading pattern in App.tsx (the `w-12 h-12 border-2 border-primary border-t-transparent rounded-full animate-spin` div). Include "Checking access..." text below.
6. If authorized (`isCoach()` returns true): render `children`
7. If unauthorized (`isCoach()` returns false): render `<Navigate to="/" replace />`
8. Accept `children: React.ReactNode` as prop
9. Export as named export: `export function CoachGuard`
10. Handle the case where `isCoach()` throws (network error) — treat as unauthorized and redirect with a toast warning. Import `toast` from `@/stores/toastStore` (direct, not barrel).

The component should be simple — approximately 40-60 lines. Do NOT add it to any barrel export files (to keep coach code out of client bundles).

Example structure:
```tsx
import { useState, useEffect } from 'react'
import { Navigate } from 'react-router-dom'
import { isCoach } from '@/lib/supabase'
import { toast } from '@/stores/toastStore'

interface CoachGuardProps {
  children: React.ReactNode
}

export function CoachGuard({ children }: CoachGuardProps) {
  const [status, setStatus] = useState<'loading' | 'authorized' | 'unauthorized'>('loading')

  useEffect(() => {
    isCoach()
      .then(result => setStatus(result ? 'authorized' : 'unauthorized'))
      .catch(() => {
        toast.warning('Unable to verify coach access')
        setStatus('unauthorized')
      })
  }, [])

  if (status === 'loading') {
    return (/* loading spinner */)
  }

  if (status === 'unauthorized') {
    return <Navigate to="/" replace />
  }

  return <>{children}</>
}
```
