**Critical Path Testing (follow checklist order):**

2. **Authentication:** Create new account OR sign in with existing

4. **Home Screen:** Verify dashboard elements display correctly
5. **Workout Flow:** Start workout, log sets, complete workout
6. **Macro Tracking:** Log macros, search food, save a meal
7. **Check-in & XP:** Do daily check-in, verify XP awards
8. **Settings:** Test weight logging, data export

**PWA Testing:**
- iOS: Open in Safari, tap Share > Add to Home Screen, launch standalone
- Android: Open in Chrome, look for install prompt or use menu > Add to Home Screen

**Offline Testing (5 scenarios):**
1. Enable DevTools Network > Offline, reload app
2. Log workout while offline, verify saved locally message
3. Return online, verify data syncs
4. Start workout online, go offline mid-workout, complete
5. Try food search offline, verify graceful handling

**Document bugs in 01-01-BUG-LOG.md as you find them.**
