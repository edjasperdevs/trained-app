# Coach Dashboard Enhancements

## Client Management

### Client Organization
- Tags/labels for client categorization (beginner, competitor, etc.)
- Custom client groups (morning clients, online clients)
- Client search and filtering
- Bulk actions (message all, export all data)

### Client Onboarding
- Custom onboarding forms per coach
- Intake questionnaire templates
- Automatic goal setting based on intake
- Welcome message automation

### Client Lifecycle
- Client status tracking (active, paused, churned)
- Renewal reminders
- Win-back campaigns for inactive clients
- Client tenure milestones

---

## Communication

### In-App Messaging
- Direct message to individual clients
- Broadcast messages to all clients
- Message templates for common communications
- Read receipts

### Check-In System
- Weekly check-in prompts (customizable questions)
- Photo submission for progress pics
- Client notes/journal visible to coach
- Structured feedback forms

### Notifications
- Alert when client misses X days
- Notification when client hits milestone
- Weekly digest of client activity
- Urgent alerts (client flagged something)

---

## Programming & Prescriptions

### Workout Programming
- Create workout templates
- Assign programs to clients
- Progressive overload suggestions
- Deload week scheduling

### Macro Prescriptions
- Set client macro targets from dashboard
- Adjust targets based on progress
- Macro cycling (high/low days)
- Refeed day scheduling

### Program Library
- Save and reuse programs
- Program categories (strength, hypertrophy, cut)
- Share programs between coaches (marketplace)

## Analytics & Reporting

### Client Analytics
- Individual client progress reports
- Compliance scoring and trends

### Business Analytics
- Total active clients

- Revenue tracking (if payments integrated)

### Exportable Reports
- Client progress PDF for client
- Aggregate reports for coach records
- Tax/business documentation
