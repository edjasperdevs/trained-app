# Social Features

## Leaderboards

### Global Leaderboards
- Weekly XP leaderboard (resets Sunday)
- Monthly streak leaderboard
- All-time rank leaderboard
- Filter by: friends only, similar level range, geographic region

### Leaderboard Privacy
- Opt-in visibility (default private)
- Anonymous mode (show rank but hide username)
- Display name customization

### Rewards
- Weekly top 10 badge
- Monthly podium recognition
- Seasonal titles/flair

---

## Friends System

### Friend Management
- Add friends via username or invite link
- Friend request approval flow
- Block/remove friend functionality
- Friend limit (start with 50, expand later)

### Friend Activity
- Activity feed: friend workouts, achievements, level-ups
- "Cheer" reaction on friend activities
- Friend comparison (your stats vs theirs, opt-in)

### Friend Discovery
- Suggest friends from contacts (permission-based)
- "People you may know" based on similar activity patterns
- Public profiles for influencers/coaches

---

## Challenges

### Daily/Weekly Challenges
- System-generated challenges (log 5 workouts this week)
- Streak challenges (maintain 7-day streak)
- Macro challenges (hit protein target 5 days)

### Friend Challenges
- 1v1 challenges: who logs more workouts this week
- Challenge templates: workout race, streak showdown, macro master
- Wager system (badges, bragging rights)

### Community Challenges
- Creator-hosted challenges (coach creates for their clients)
- Time-limited events (New Year challenge, Summer shred)
- Participation badges for joining

---

## Groups/Clans

### Group Features
- Create/join groups (gym buddies, online communities)
- Group chat (simple text, no media initially)
- Group leaderboards (internal competition)
- Group challenges (team vs team)

### Group Roles
- Owner: full control
- Admin: moderate members
- Member: participate

### Group Discovery
- Public groups directory
- Invite-only private groups
- Group size limits (start with 25)

---

## Social Profile

### Profile Customization
- Bio/about section
- Featured badges (pin up to 3)
- Avatar showcase
- Workout split display

### Profile Privacy
- Public: anyone can view
- Friends-only: only approved friends
- Private: no one can view

### Profile Stats
- Total workouts logged
- Current streak
- Longest streak
- Rank achieved
- Member since date
